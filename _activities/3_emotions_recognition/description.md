# Descripcion

Esta actividad trato de armar un dataset para deteccion de emociones en este caso (**Feliz**, **Enojado** y **Triste**).

Para lograr un buen dataset se utilizo la pagina de kaggle.com en la cual se pueden encontrar bastantes data sets armados.

Logre conseguir uno en el cual aproximadamente cada clase tenia entre 5,000 y 7,000 que fueron empleadas para entrenar los 3 modelos vistos.

Los resultados no fueron buenos ya que en mi caso puede que el dataset no haya sido lo suficientemente bueno ya que con ningun modelo reconocia emociones de forma clara y precisa tenia mucha variacion y nunca te dectaba fijo una emocion.
